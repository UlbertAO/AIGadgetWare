# EncrytDecrypt

This project is a small C# console application that demonstrates how to encrypt and decrypt text using AES encryption in .NET. It takes a sample string, encrypts it, and then decrypts it back so you can see the full process in action.

## What this project is about

It is a simple example of secure text handling in C#. The app shows how to:

- encrypt plain text
- derive an encryption key
- convert the encrypted result into a safe string format
- decrypt it back to the original value

## How it can benefit you

This project is useful for learning basic encryption concepts in .NET and for understanding how simple encryption can be implemented in a console app. It can also serve as a starting point for adding encryption to small applications.

## How to run

Make sure you have .NET 6 installed, then run the following commands from the project folder:

```bash
dotnet build
dotnet run
```

The program will print the original text, the encrypted output, and the decrypted result.
